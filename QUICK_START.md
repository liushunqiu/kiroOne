# Kiro One - 快速启动指南

## ✅ 问题已解决

项目已经可以正常运行了!主要问题是:
1. **缺少 Tauri CLI** - 已通过 npx 解决
2. **需要正确的启动方式** - 项目是 Tauri 桌面应用,不能单独运行前端

## 🚀 启动方式

### 方式 1: 双击启动 (推荐)
直接双击项目根目录下的 `start-dev.bat` 文件

### 方式 2: 命令行启动
```bash
cd D:\program\kiroOne
npm run tauri:dev
```

### 方式 3: 使用 npx
```bash
cd D:\program\kiroOne
npx tauri dev
```

## 📋 环境要求

已经确认以下环境已安装:
- ✅ Node.js (npm 依赖已安装)
- ✅ Rust 1.94.1
- ✅ Tauri CLI (通过 npx)

## 🎯 功能说明

应用启动后,您应该能看到:

1. **仪表盘** - 显示统计信息
2. **账号管理** - 添加、编辑、导入导出账号
3. **API 网关** - 配置和启动网关服务
4. **供应商设置** - 管理和切换 AI 供应商

## ⚠️ 注意事项

1. **首次启动可能较慢** - 需要编译 Rust 代码,请耐心等待 (1-2 分钟)
2. **数据保存在内存中** - 当前版本重启应用后数据会清空
3. **端口占用** - 如果 8710 端口被占用,请在网关设置中更换端口

## 🔧 常见问题

### Q: 启动后窗口没有显示?
A: 检查任务栏是否有 "AI Unified Manager" 窗口,可能被其他窗口遮挡

### Q: 编译失败?
A: 确保网络连接正常,首次启动需要下载 Rust 依赖

### Q: 如何完全停止应用?
A: 关闭应用窗口,或在命令行中按 Ctrl+C
